javascript高级程序设计之变量、作用域和内存

前端

基本类型和引用类型的特点

  1. 基本类型值在内存中占据固定大小的空间，因此保存在栈中
  2. 引用类型值是对象，保存在堆中
  3. 包含引用类型值的变量实际上包含的并不是对象本身，而是一个指向该对象的指针
  4. 从一个变量向另一个变量复制引用类型的值，复制的其实是指针，因此两个变量最终都指向同一个对象

传递参数

划重点：在函数内部重写obj的时候，这个变量的引用就已经是个局部对象了，并且这个会在函数执行完毕后会被立即销毁，这个例子也证明了对象是按值传递的

内存管理

一旦数据不再使用，最好通过将其值设置为null来释放其引用，这个做法适用于大多数全局变量以及全局变量对象的属性，局部变量会在离开执行环境时自动解除引用。

前言

javascript高级程序设计这本书，我在读大学的时候就通读过一遍。如今在工作之余，再回头仔细的阅读一遍，记录一些容易忘记和忽略的知识点。其中也会掺杂一些其他乱七八糟的东西，比如代码规范、设计模式以及我和我的工作等。

数据类型

ECMAScript有五种基本数据类型：Undefined、Null、Boolean、Number和String，还有一种复杂数据类型Object。

typeOf操作符

typeOf可能返回的值：

  • undefined：未定义
  • boolean：布尔值
  • string：字符串
  • number：数值
  • object：对象或null
  • function：函数

Null类型

从逻辑上来讲，null表示一个空对象指针，在工作中常用来解除变量和对象之间的引用关系，方便浏览器进行垃圾回收。

Boolean类型

转换规则：

  • 任何非空字符串 => true
  • 任何非零数字（包括无穷大） => true
  • 0和NAN => false
  • 除null以外的任何对象（包括空对象和空数组） => true
  • undefined => false

任意两个NaN都不相等

常用技巧：在工作中，经常会使用!!将变量转换为boolean类型

Number类型

  1. 科学计数法
  2. 浮点数精度
  3. 数值范围
  • 负无穷：Number.NEGATIVE或Number.MIN_VALUE
  • 正无穷：Number.POSITIVE_INFINITY或Number.MAX_VALUE
  • 判断数字是否在javascript的数值范围内：
  1. NaN
    isNaN()函数被用来判断一个变量是否不是数字，但是这个方法得到的结果并不准确。

    String类型

  3. String() 方法可以将任何类型的变量转换为字符串，转换时优先使用toString()方法，如果是null则返回"null"，如果是undefined则返回"undefined"。

git常用操作

开发工具

git stash用于保存和恢复工作进度

  • git stash：保存当前的工作进度。会分别对暂存区和工作区的状态进行保存。
  • git stash save "meassage"：这是第一条命令的完整版，在保存的时候给本次进度打上标记信息。
  • git stash list：显示保存的进度列表。
  • git stash pop [--index] [\<stash>]：如果不使用任何参数，恢复最新保存的工作进度，并将恢复的工作进度从进度列表中清除。如果提供参数（来自git stash list显示的列表），则从该stash中恢复。恢复完毕也将从进度列表中删除。选项–index除了恢复工作区外的文件外，还尝试恢复暂存区。
  • git stash apply [--index] [\<stash>]：除了不删除恢复的进度外，其余和git stash pop命令一样。
  • git stash clear：删除所有存储的进度

git update-index --assume-unchanged的用法

  1. 你正在修改一个巨大的文件，你先对其 git update-index –assume-unchanged，这样 Git 暂时不会理睬你对文件做的修改；
  2. 当你的工作告一段落决定可以提交的时候，重置改标识：git update-index –no-assume-unchanged，于是 Git 只需要做一次更新，这是完全可以接受的了；
  3. 提交+推送；
