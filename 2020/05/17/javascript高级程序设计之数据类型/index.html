javascript高级程序设计之数据类型

前端

前言

javascript高级程序设计这本书，我在读大学的时候就通读过一遍。如今在工作之余，再回头仔细的阅读一遍，记录一些容易忘记和忽略的知识点。其中也会掺杂一些其他乱七八糟的东西，比如代码规范、设计模式以及我和我的工作等。

数据类型

ECMAScript有五种基本数据类型：Undefined、Null、Boolean、Number和String，还有一种复杂数据类型Object。

typeOf操作符

typeOf可能返回的值：

  • undefined：未定义
  • boolean：布尔值
  • string：字符串
  • number：数值
  • object：对象或null
  • function：函数

Null类型

从逻辑上来讲，null表示一个空对象指针，在工作中常用来解除变量和对象之间的引用关系，方便浏览器进行垃圾回收。

Boolean类型

转换规则：

  • 任何非空字符串 => true
  • 任何非零数字（包括无穷大） => true
  • 0和NAN => false
  • 除null以外的任何对象（包括空对象和空数组） => true
  • undefined => false

任意两个NaN都不相等

常用技巧：在工作中，经常会使用!!将变量转换为boolean类型

Number类型

  1. 科学计数法
  2. 浮点数精度
  3. 数值范围
  • 负无穷：Number.NEGATIVE或Number.MIN_VALUE
  • 正无穷：Number.POSITIVE_INFINITY或Number.MAX_VALUE
  • 判断数字是否在javascript的数值范围内：
  1. NaN
    isNaN()函数被用来判断一个变量是否不是数字，但是这个方法得到的结果并不准确。

    String类型

  2. toString() 方法用于将变量转换为字符串，null和undefined没有该方法。
  3. String() 方法可以将任何类型的变量转换为字符串，转换时优先使用toString()方法，如果是null则返回"null"，如果是undefined则返回"undefined"。
